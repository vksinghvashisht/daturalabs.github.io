## Deploy to GitHub Pages (one-time setup)

### Step 1: Create GitHub repo
1. Go to github.com/daturalabs
2. Create new repository named exactly: `daturalabs.github.io`
3. Set visibility: **Public**
4. Do NOT initialize with README

### Step 2: Push these files
```bash
cd /path/to/this/folder
git init
git add .
git commit -m "Initial deploy"
git remote add origin https://github.com/daturalabs/daturalabs.github.io.git
git push -u origin main
```

### Step 3: Point GoDaddy DNS to GitHub Pages
In GoDaddy DNS settings for daturalabs.com, add these records:

| Type  | Name | Value                   | TTL |
|-------|------|-------------------------|-----|
| A     | @    | 185.199.108.153         | 600 |
| A     | @    | 185.199.109.153         | 600 |
| A     | @    | 185.199.110.153         | 600 |
| A     | @    | 185.199.111.153         | 600 |
| CNAME | www  | daturalabs.github.io.   | 600 |

### Step 4: Set custom domain in GitHub
1. In repo → Settings → Pages
2. Custom domain: `daturalabs.com`
3. Check "Enforce HTTPS"
4. Wait 10-30 min for DNS propagation

Site is live at daturalabs.com. ✓
